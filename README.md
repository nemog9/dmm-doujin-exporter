# DMM Doujin Exporter

DMMマイライブラリの同人作品（ボイス・コミック・CG）をCSVにエクスポートするChrome拡張機能。

エクスポートしたデータは以下のリポジトリでいい感じに閲覧できます

https://github.com/nemog9/doujin-shelf

## 機能

- マイライブラリをスクロールして作品情報を自動収集
- 声優（ボイス作品）・作者（コミック・CG作品）を各作品の詳細ページから取得
- 既存CSVとのマージ出力（前回の続きから差分のみ取得）
- 重複検知モード（デフォルト）と全件探索モードの切り替え

## 出力形式

UTF-8 BOM付きCSV

```
タイトル,サークル名,声優,URL,サムネイル
```

| カラム | 内容 |
|--------|------|
| タイトル | 作品名 |
| サークル名 | サークル名 |
| 声優 | 声優名（ボイス）または作者名（コミック・CG）、複数はカンマ区切り |
| URL | マイライブラリの作品詳細URL |
| サムネイル | サムネイル画像URL |

## インストール

1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeで `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をON
4. 「パッケージ化されていない拡張機能を読み込む」からフォルダを選択

## 使い方

1. DMMにログインし、[マイライブラリ](https://www.dmm.co.jp/dc/-/mylibrary/)を開く
2. 拡張機能アイコンをクリック
3. （任意）前回エクスポートしたCSVを読み込む → 差分のみ取得してマージ出力される
4. 「取得件数」を設定（0で全件）
5. 「取得開始」をクリック

### 全件探索モード

チェックを入れると、重複を検知しても最後までスクロールし続ける。  
通常は不要だが、ライブラリの途中に未取得の作品がある場合などに使用する。

### ファイル名

前回のCSVを読み込んだ場合、そのファイル名のベース部分（例: `my-library`）に現在の日時を付けて保存される。

```
my-library-20260501-153000.csv
```

CSVなしで開始した場合は `doujin-library-YYYYMMDD-HHMMSS.csv` で保存される。

## 動作の流れ

```
① スクロールフェーズ
   マイライブラリページを自動スクロールして作品一覧を収集
   （デフォルト: 既存IDと重複した時点で停止）

② 声優・作者取得フェーズ
   新規作品のみ詳細ページをfetchして声優/作者を取得

③ CSV出力
   既存CSVの内容 + 新規作品をマージしてダウンロード
```

## 注意事項

- DMMのマイライブラリページを開いた状態で使用してください
- 作品数が多い場合、取得に時間がかかります（1作品あたり約0.5秒）
- DMMのページ構造変更により動作しなくなる場合があります
